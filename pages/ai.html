Course Overview

The Artificial Intelligence course provides a strong foundation in AI, covering machine learning, deep learning, NLP, and neural networks. Work on real-world projects using Python, explore algorithms, and gain industry-ready AI skills.

📚 Curriculum

Week 1–2: Introduction to AI & Python

  • Basics of AI, ML, DL & applications
  • Python refresher: syntax, data types, functions
  • Jupyter Notebook & Anaconda setup

Week 3: Machine Learning with Scikit-learn

  • Supervised vs Unsupervised Learning
  • Classification & Regression models
  • Model evaluation techniques

Week 4–5: Deep Learning with TensorFlow

  • Neural Networks & Backpropagation
  • Building models with Keras
  • Image classification project

Week 6: Natural Language Processing (NLP)

  • Text preprocessing techniques
  • Sentiment analysis with NLTK
  • Intro to chatbot development

Week 7–8: AI Projects & Career Prep

  • Mini capstone project (choose AI problem)
  • Git & GitHub version control
  • Resume building & interview prep
